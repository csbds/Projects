## 📁 Projects

### 1. Apple Fitness XML — Data Cleaning

#### Overview
Processed and cleaned Apple Fitness export data stored in XML format to prepare it for forecasting and analytics workflows.

#### Key Tasks
- Parsed XML health and fitness records
- Removed duplicate entries
- Handled missing values and inconsistent timestamps
- Standardized workout and activity metrics
- Created analysis-ready datasets

#### Technologies
`Python` `Pandas` `XML ElementTree`

#### Outcome
Produced a reliable dataset suitable for trend analysis and predictive modeling.

---

### 2. Apple Music Playlists XML → CSV — Data Transformation

#### Overview
Built a data transformation pipeline that converts Apple Music playlist exports from XML format into structured CSV files.

#### Key Tasks
- Extracted playlist metadata
- Parsed song information
- Converted nested XML structures into tabular datasets
- Automated CSV generation
- Visualized data trends

#### Technologies
`Python` `Pandas` `XML Parsing`

#### Outcome
Enabled efficient analysis of listening habits, playlist composition, and music consumption trends.

---

### 3. Mac Application Malfunction Analyzer — Data Analysis

#### Overview
Analyzed macOS application logs and diagnostic data to identify software malfunctions and crashes.

#### Key Tasks
- Processed application malfunction records
- Investigated failure frequency and error patterns
- Created analysis-ready datasets

#### Technologies
`Python` `Pandas` `Jupyter Notebook`

#### Outcome
Produced a reliable dataset suitable for providing insights into application stability and recurring failure scenarios.

### 4. App Screen-Time Tracker

#### Key Tasks

### 5. Disk Usage Forecasting — Predictive Analytics

#### Overview
